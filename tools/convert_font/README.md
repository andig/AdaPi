convert_font
============

Truetype to ASCII bitmap font conversion tool
requires FontBuilder bitmap font generator:
 
	https://github.com/andryblack/fontbuilder

Designed for use with the adafruit SSD1306 OLED.
Produces bitmapped fonts designated for the 
gaugette python library:

	https://github.com/guyc/py-gaugette
